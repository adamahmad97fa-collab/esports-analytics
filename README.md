# Esports Match Analytics

## Overview
This project analyses competitive esports match data to understand team performance and map strengths. It shows how raw match data can be stored, analysed, and turned into insights that support preparation and strategic decision-making.

## Tech Stack
- Python  
- Pandas  
- SQLite  
- Matplotlib  

## Data
The project uses structured match-level data stored in a SQLite database. The data is cleaned and standardised before analysis so results can be compared consistently across matches and maps.

## What the Project Does
- Calculates overall win rate  
- Breaks down win rate by map  
- Identifies strongest and weakest maps  
- Produces visualisations to support strategic focus and map veto decisions
